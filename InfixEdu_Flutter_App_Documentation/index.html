    Payment Gateway

  • Paytm
  • Paypal
  • Gpay and Apple pay
  • Razor

    • Paytm

    Go to https://developer.paytm.com/ and create an account.

    • Then fetch merchant id and keys from https://dashboard.paytm.com/next/apikey

    If you want you may go through this link paytm

    You should have a file from the path payment gateway=>paytm=>server=>config => which named appConfig.js replace below keys with yours

    • Then in flutter project infixEdu=>lib=>paymentGateway=>paytm=>settings.dart , replace url where you deploy your backend server code or if you work on localhost you have to node.js install and start npm server , replace below url with your url along with ip address (example:http//your-computer-ip-address:3000/api/v1/paytm/initialPayment)

    Paypal

    Go to paypal account and create sandbox account for individual and business(test purpose)https://developer.paypal.com/

    You need business account for receive money and individual account for pay money . You can login sandbox account through this business and individual email and password

    • Then fetch client id and client secret key from your business account
    • You should have a file from the path payment gateway=>paypal=>which named app.js replace below keys with yours
    • You should have migrate mode after test with 'sandbox' to 'live' (you need to change client id and client secret key as well) if you want to get real payment in your business account

    If you want you may go through this link paypal with nodejs

    Then you need to change apiUrl from your flutter app for that you need to install nodeJs and start nodejs server , see below image carefully at line 18 .

    • In flutter project infixEdu=>lib=>paymentGateway=>payapal=>PaypalHomeScreen.dart , replace url where you deploy your backend server code or if you work on localhost you have to node.js install and start npm server , replace below url with your url along with ip address (for example : http//'your-computer-ip-address:3000/)

    Square API For Gpay And Apple Pay

    Go to https://developer.squareup.com/us/en and create an account . Click on the new application to bring up the Square application settings pages.You will need the Application ID from the Credentials page to configure In-App Payments SDK

    If you want you may go through this link Square

    You should have Application ID,Location ID and Apple Merchant ID to get payment from Gpay ,Apple Pay and Master Card,Credit Card

    • Then in flutter project infixEdu=>lib=>Utils=>config.dart , replace respective keys which are getting from squareup developer account

    Razor Payment Gateway

    Create an account to Razor . Select the mode (Test or Live) for which you can to generate the API key. You have to generate separate API Keys for the test and live modes. No money is deducted from your account in test mode.

    If you want you may go through this link razor documentation

    You find the Key ID from your razor dashboard=>settings section

    • Then in flutter project infixEdu=>lib=>paymentGateway=>RazorPayHome.dart , replace respective keys which are getting from razor developer account

    Add Firebase push notification to your Flutter app(android/ios)

    Before add firebase to your flutter app, you need to create a firebase project to connect to your app.

    At first create project at firebase

    Gradle file (android/build.gradle),add rules to include the Google Services Gradle plugin.Check that you have Google’s Maven repository

    classpath 'com.google.gms:google-services:4.3.3'

    In your module (app-level) Gradle file (usually android/app/build.gradle), apply the Google Services Gradle plugin at bottom.

    apply plugin: 'com.google.gms.google-services'

    In your module (app-level) Gradle file (usually android/app/build.gradle), apply the Google Services Gradle plugin at bottom.

    apply plugin: 'com.google.gms.google-services'

    In your firebase application dashboard => Cloud Messaging => general create config file for android and ios . You have to provide package name for android and bundle identifier for ios. Download google-service.json then paste into android=>app folder , Download GoogleService-info.plist then paste into your ios=>Runner folder
